# Java基础知识
## 一、JDK、JRE、JVM之间的关系
JDK = JRE  + java开发工具
JRE = JVM  + 核心类库

## 二、java编写7个规范
1. 类、方法的注释，使用javadoc注释，即文档注释
2. 非javadoc注释，对代码说明，说明如何修改，注意事项
3. 使用tab，整体将代码右移，shift+tab，整体将代码向左移动
4. 运算符和 = 两边给空格
5. 源码文件用 utf-8 编码
6.行宽字符不超过 80 
7. 行尾风格和次行风格（推荐）

## 三、浮点数在机器中的存放形式
1. 浮点数=符号位+指数位+尾数位
2. 尾数部分可能丢失造成精度损失
